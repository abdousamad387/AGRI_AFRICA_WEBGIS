## 🌍 AgriAfrica WebGIS — Interactive Agricultural Atlas

A premium, modern, and fully responsive WebGIS platform for visualizing, analyzing, and comparing agricultural, climate, and socio-economic indicators across Africa (2000–2024).

### 🚀 Features
- **Interactive Map**: Fast, beautiful choropleth with 5 basemaps (Dark, Satellite, Terrain, Light, OSM)
- **Indicators**: 35+ indicators, 7 categories (production, climate, trade, security, finance, social, sustainability)
- **Advanced Statistics**: Descriptive stats, regression, ANOVA, spatial autocorrelation (Moran’s I, Getis-Ord G*)
- **Spatial Tools**: Distance & area measurement
- **Panels**: Country profile, ranking, data table, compare, statistics
- **Resizable Panels & Sidebar**: Drag to resize, ultra-responsive
- **i18n**: English/French toggle, full translation
- **Export**: Charts exportable as images
- **Premium UI**: Glassmorphism, animated gradients, Bootstrap Icons, Inter/JetBrains Mono fonts
- **Open Source**: MIT License

### 🛠️ Tech Stack
- **Frontend**: React 18, TypeScript, Vite
- **Mapping**: Leaflet, react-leaflet
- **Charts**: Recharts
- **Styling**: Custom CSS, Bootstrap Icons, Google Fonts
- **Data**: 14 JSON files + Africa GeoJSON

### 🌐 Demo
[Live Demo (GitHub Pages)](https://abdousamad387.github.io/AGRI_AFRICA_WEBGIS/)

### 📦 Getting Started
```bash
# Clone
 git clone https://github.com/abdousamad387/AGRI_AFRICA_WEBGIS.git
 cd AGRI_AFRICA_WEBGIS

# Install dependencies
 npm install

# Start dev server
 npm run dev

# Build for production
 npm run build

# Deploy to GitHub Pages
 npm run deploy
```
